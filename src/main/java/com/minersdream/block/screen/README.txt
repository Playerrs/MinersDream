For create a new block

-> create a code in main/java/minersdream/block/MobBlocks file

-> create a new file in assets/minersdream/blockstates folder > namebrock.json

-> create a new file in assets/minersdream/models/blocks folder > nameblock.json

-> create a new file in assets/minersdream/models/item folder > nameblock.json

-> create 2 new line in assets/minersdream/lang/file  ex "item.minersdream.raw_asnium": "Asnium bruto", and  "tooltip.minersdream.raw_asnium": "Usado para receitas mais complexas",
